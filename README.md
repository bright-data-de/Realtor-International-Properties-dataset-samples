Dieses **Realtor International Properties Dataset-Beispiel** umfasst **1.000 Datensätze** und wurde mit der **Bright Data API** extrahiert.

## Datenpunkte

| Feldname | Beschreibung |
|------------|-------------|
| `url` | Die direkte URL zur detaillierten Immobilien-Anzeigenseite auf Realtor. |
| `id` | Eine eindeutige Kennung, die jeder Immobilienanzeige zugewiesen wird. |
| `title` | Der Titel oder die Überschrift der Immobilienanzeige, einschließlich Adressdetails. |
| `price` | Der angegebene Preis der Immobilie in der festgelegten Währung. |
| `currency` | Die Währung, in der der Preis der Immobilie angegeben ist. |
| `description` | Eine kurze Beschreibung oder Übersicht über die Merkmale und Ausstattungen der Immobilie. |
| `images` | URLs von Bildern, die die Immobilie zeigen, als Liste bereitgestellt. |
| `bedrooms` | Die Anzahl der in der Immobilie verfügbaren Schlafzimmer. |
| `bathrooms` | Die Anzahl der in der Immobilie verfügbaren Badezimmer. |
| `property_type` | Die Art der Immobilie, z. B. Haus, Apartment, Villa. |
| `building_size` | Die Gesamtgröße des Gebäudes in Quadratfuß oder Quadratmetern. |
| `land_size` | Die Gesamtgröße des Grundstücks, auf dem die Immobilie gebaut ist. |
| `rooms` | Die Gesamtzahl der in der Immobilie verfügbaren Zimmer. |
| `address` | Die vollständige Straßenadresse der Immobilie. |
| `country_code` | Der ISO-Code, der das Land angibt, in dem sich die Immobilie befindet. |
| `state` | Das Bundesland oder die Region, in dem bzw. der sich die Immobilie befindet. |
| `city` | Die Stadt, in der sich die Immobilie befindet. |
| `realtor_details` | Informationen über den Makler oder die Agentur, die die Immobilienanzeige verwaltet. |
| `price_history` | Historische Daten zu den Preisänderungen der Immobilie im Zeitverlauf. |
| `listing_status` | Der aktuelle Status der Anzeige, z. B. aktiv, ausstehend, verkauft. |


## Anwendungsfälle

### 1. Marktanalyse
Nutzen Sie dieses Dataset, um umfassende Marktanalysen durchzuführen und Trends sowie Preisstrategien in verschiedenen Regionen zu identifizieren.

### 2. Investitionsmöglichkeiten
Identifizieren Sie lukrative Immobilien-Investitionsmöglichkeiten durch die Analyse von Immobiliendetails und historischen Preisdaten.

### 3. Wettbewerbsvergleich
Verwenden Sie das Dataset, um Immobilienanzeigen zu vergleichen und die Wettbewerbspositionierung auf dem Immobilienmarkt zu bewerten.
